# Linux Commands

### Updated everyday

`whoami`: Gives detail about the user login as. <br>
`whoami `

`man`: Tells about the commands. <br>
`man <command name>`

`clear`: Clear the terminal. <br>
`clear`

`pwd`: Tells the location we currently in. <br>
`pwd`

`ls`: List the content of a folder. <br>
`ls <file name>`
